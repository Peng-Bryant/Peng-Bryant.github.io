Research Experience and Selected Projects

My current research and project topics cover motion planning, assistive robotics, and autonomous system design. My long-term research goal is to develop intelligent robots that can infer and interact with the dynamic and open world in long-horizon tasks.

Development of a Vision-Guided Autonomous UAV for Precision Targeting and Shooting of Moving Objects

1. High-speed flight capabilities for fast moving objects targeting.
2. Vision-based automatic aiming and tracking system with high accuracy and robustness against interference.
3. Adaptive control algorithms for improved resilience and adaptability during flight and shooting operations.
A Versatile Mobile Robot With a Visual-aided Suction Gripper for Object Transport

This Project won the Second Prize in RoboMaster2022 Engineering-Mining-Challenge.
1. Three-DOF suction gripper: Enhanced grasping flexibility and precision.
2. Multifunctionality: Versatile robot for various tasks and industries.
3. Vision-based alignment: Improved accuracy and efficiency in positioning.
Image-Based Visual Servoing for Tracking Dynamic Objects with Unknown Velocities using a Quadruped Manipulator

1. Decoupling visual error into linear/angular components with spherical projection model.
2. Robust super-twisting observer for dynamic object velocities estimation without depth infomation.
3. Model Predictive Controller joint torques control for quadruped manipulator to track dynamic targets.
Visualizing Pandemic Simulator: A Graphical User Interface for Simulating and Analyzing Epidemic Dynamics

This project is developed by C++ and EasyX to visualize epidemic spread with analysis of the effects of various factors using Control Variable Method.
1. Interactive visualization interface.
2. Improved population modeling using a normal distribution model.
3. Optimized simulation performance with quadtree collision detection algorithm.
Design and Control of a Hybrid Legged-Wheeled Robot for Enhanced Terrain Adaptation and Efficiency Based on LQR Control

1. Hybrid legged-wheeled robot design for improved mobility and terrain adaptability.
2. Integration of LQR and VMC control methods for enhanced control performance.
3. Optimization of robot parameters for optimal efficiency and robustness.
